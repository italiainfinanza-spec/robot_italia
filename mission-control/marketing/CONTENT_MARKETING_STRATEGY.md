## ğŸ› ï¸ Technical Implementation

### On-Page SEO Checklist

For every piece of content:
- [ ] Primary keyword in H1 (natural placement)
- [ ] Secondary keywords in H2s
- [ ] Meta title < 60 chars with primary keyword
- [ ] Meta description < 155 chars with CTA
- [ ] URL slug contains primary keyword
- [ ] Internal links to/from pillar pages (min 3 per article)
- [ ] Alt text for all images
- [ ] Schema markup: Article, FAQPage where applicable
- [ ] Table of contents for 1,500+ word articles
- [ ] Social sharing images (1200x630px)

### Content Optimization Schedule

| Content Age | Action |
|-------------|--------|
| 30 days | Review performance, optimize underperformers |
| 90 days | Refresh outdated stats, expand if ranking well |
| 6 months | Major update, re-publish with new date |
| 12 months | Consolidate or redirect if underperforming |
